# Priced Out
### Why is Chinatown becoming less Chinese?

An interactive scrollytelling visualization documenting the displacement of Asian residents from Boston's Chinatown (ZIP 02111) through rising rents, 2020–2026.

---

## Data Sources

| Data | Source |
|---|---|
| Rent (1BR median, ZIP 02111) | [RentHop](https://www.renthop.com/average-rent-in/02111-boston-ma) |
| Asian population 2020 & 2025 | ACS / BPDA Boston Neighborhood Population Estimates |
| Asian population 2021–2024 | Interpolated between 2020 & 2025 ACS anchors |
| Asian population 2026 | Extrapolated by extending 2020–2025 decline slope |
| Population share (2010, 2020) | BPDA via The Scope Boston |
| 80% housing insecurity stat | MIT Displacement Research Action Network & Chinese Progressive Association |
| ~50% income spent on rent | MIT DRAN & Chinese Progressive Association |
| 10+ year affordable housing waitlist | The Scope Boston |
| All resident quotes | [The Scope Boston — Cao, Chen & Hu (April 8, 2025)](https://thescopeboston.org/9791/housing/why-is-chinatown-becoming-less-chinese/) |
